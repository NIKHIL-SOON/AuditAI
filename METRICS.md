# North Star Metrics

## Primary North Star
**Total Annual Savings Found:** The cumulative dollar amount of annual savings identified by the AuditAI engine across all users. This proves the aggregate value created by the product.

## Key Performance Indicators (KPIs)
1. **Audit Completion Rate:** Percentage of users who start the form and successfully generate an audit result.
2. **High-Value Lead Ratio:** Percentage of completed audits that cross the $500/mo savings threshold (triggering the Credex consultation CTA).
3. **Viral Coefficient (Share Rate):** Number of times the shareable audit URL (`/share/[id]`) is visited per completed audit.
4. **Consultation Conversion Rate:** Percentage of high-value leads that actually book a meeting via the CTA.
